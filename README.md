      BOT WHATSAPP - TUAN MUDA
                  ☠️
                  
Silakan ikuti panduan di bawah ini untuk memulai.

--- CARA INSTALL ---
1. Buka Termux.
2. Masuk ke folder bot ini.
3. Ketik perintah:
4. git clone https://github.com/iksan757/wabot-tuan-muda.git
5. cd wabot-tuan-muda
   chmod +x install.sh
   ./install.sh
6. Tunggu sampai semua proses selesai.

--- CARA MENJALANKAN ---
1. Ketik perintah: node index.js
2. Masukkan nomor WhatsApp Anda (Format 628xxxxxxxxxx).
3. Masukkan KODE PAIRING 8 digit ke WhatsApp Anda (Perangkat Tertaut).
4. Jika sudah muncul status "CONNECTED", bot siap digunakan.

--- TENTANG FOLDER 'auth_info' ---
- Folder 'auth_info' akan otomatis muncul setelah Anda login.
- Folder ini berisi sesi login WhatsApp Anda.
- JANGAN MENGHAPUS folder ini jika Anda ingin bot otomatis login
  tanpa harus memasukkan kode pairing lagi.
- JANGAN MEMBERIKAN folder ini kepada orang lain karena mereka
  bisa mengakses WhatsApp Anda tanpa izin.

--- DAFTAR PERINTAH (COMMAND) ---
Ketik perintah ini di chat WhatsApp:

> seusaikan dengan kode negara anda!
> !menu
  (Melihat semua daftar menu)

> !kirim [nomor_tujuan] [pesan]
  Contoh: !kirim 628xxxxxxxx Halo bos

> !spam [nomor_tujuan] [jumlah] [pesan]
  Contoh: !spam 62xxxxxxxxx 10 P
